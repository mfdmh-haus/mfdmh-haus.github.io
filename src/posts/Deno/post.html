Deno!

When I started working with Deno a few days ago, latest was v1.30.0. Now it's 1.30.3! Batteries included: TypeScript support, Package manager, Bundler, Watcher Secure by default Surprise! Deno Deploy's Deno is not the same as your local Deno. It's a subset of the stable APIs. Favors Web standards over Node.js conventions: Supports but doesn't force you to use import maps as an alternative to NPM's package.json dependencies field It does not have a built-in analog for devDependencies or peerDependencies. For something like that, they recommend this approach.
